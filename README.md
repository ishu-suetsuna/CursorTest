# Museum AI Guide - AI音声ガイドアプリ

美術館・博物館の展示物に関するAI音声ガイドのWEBアプリケーションです。

## 🎯 機能

### ログイン画面 (`login.html`)
- ユーザー認証
- ログイン状態の保持
- レスポンシブデザイン
- 美しいアニメーション効果

### 音声ガイド一覧 (`guide-list.html`)
- 展示物の一覧表示
- カテゴリー別フィルタリング
- 検索機能
- お気に入り登録
- ページネーション

### 音声ガイド詳細 (`guide-detail.html`)
- 展示物の詳細情報
- 音声プレイヤー
- 再生速度調整
- 文字起こし表示
- 関連作品の表示

## 🚀 使用方法

1. **ログイン画面**から開始
   - 任意のメールアドレスとパスワードでログイン
   - ログイン成功後、音声ガイド一覧に自動遷移

2. **音声ガイド一覧**で作品を選択
   - カテゴリーや並び順でフィルタリング
   - 検索で特定の作品を探す
   - お気に入りに登録

3. **詳細画面**で音声ガイドを視聴
   - 音声プレイヤーで再生・一時停止
   - 再生速度を調整
   - 作品の詳細情報を確認
